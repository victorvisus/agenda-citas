# Carpeta Utils

Funciones utilitarias reutilizables para validaciones y otras operaciones comunes.

## Archivos

- `validaciones.js` - Funciones para validar email, teléfono, servicio, etc.

## Uso

```js
import { validarEmail } from "./utils/validaciones.js";
if (validarEmail("usuario@mail.com")) {
  /* ok */
}
```
