# Carpeta JS

Contiene todo el código JavaScript de la aplicación, organizado en módulos ES6.

## Estructura

- `index.js` - Punto de entrada principal.
- `models/` - Clases de datos (e.g., Cita).
- `utils/` - Funciones utilitarias (validaciones).
- `exceptions/` - Excepciones personalizadas.
- `ui/` - Lógica de interfaz de usuario.
